#  ReadMe for Assignment 1 Question 2


## This repository covers step 2 of Module 1 Assignment. 

In this repo, I created a python file that calculates the number of finishers in a race.

Output: Percent of the race that DNF'd (Did Not Finish).

Inspired by UTMB last weekend, where over a third of racers didn't finish!
